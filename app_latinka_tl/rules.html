Правила транслітерації

М'які літери 
    створюються з використанням символа [letter] + \u030C [combining_caron]:
        д[d] + ь -> ď      молодь   -> moloď
        c[s] + ь -> š      сьогодні -> šohodni
        т[t] + ь -> ť      тьохнути -> ťoxnuty

   *крім букв які непотрібно пом'якшувати: ч, ш, щ

Сполучення приголосних з подвійними госолними
    відбувається випадання звуку й, що відповідає вимові:
        д+я[ja] -> ďa  дякую -> ďakuju
        к+ю[ju] -> ǩu  кювет -> ǩuvet

   *виключення, слова де використовується апостроф для твердного звуку:
        м'ясо -> mjaso

Кирилична буква ж 
    замінюється на латинський фонетичний символ Ezh:
        ж -> ʒ   Ж -> Ʒ
        наприклад: жарити -> ʒarutu

Сиволи для букв ч, ш, щ
    створюються за допомогою альтернативного вводу[Alt Gr] літер 'c' та 's'
        ч     -> ć
        ш     -> ś
        щ[шч] -> ść

Подвійність літери в 
    замінюється на латинську w:
        вв -> w     ввічливість -> wićlyvisť
